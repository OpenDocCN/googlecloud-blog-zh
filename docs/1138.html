云事件

CloudEvents是由云本地计算基金会的无服务器工作组发起的，目标是标准化事件发布者如何描述他们的事件。目前，计划仍在进行中，规范还没有稳定下来；许多云服务提供商和开源项目已经宣布了他们采用该规范的计划，包括 Knative Eventing 和 Azure 。

注

你可以在这里 阅读最新发布的规范(0.3)。

CloudEvents绑定有助于跨应用、服务和设备传输事件。例如，您可以将事件绑定到JSON格式，或者将其映射到HTTP请求。
