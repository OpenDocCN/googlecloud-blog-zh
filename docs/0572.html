设置

Firebase建议你用Node.js v.6.11.5进行本地开发，但是因为我们要用TypeScript写所有东西，让transpiler变魔术，所以我们将使用Node.js LTS v8.10.0。它工作正常…不用担心:)

克隆git repo:

git clone git@github.com:jeremylorino/firebase-functions-unittest-cd.git

安装Node.js依赖项

npm install
