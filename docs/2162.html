数据库

在数据库方面已经发布了多项产品:

  • Cloud SQL now 支持预览版中PostgreSQL和SQL Server的就地升级。
  • beta 中有一个全新的名为“专业云数据库工程师”的专业认证。
  • 宣布数据库迁移计划，即将现有开源和专有数据库迁移到Google Cloud数据库的方法，无论是在内部还是在云中。
  • 数据库迁移服务支持Oracle到PostgreSQL模式和数据迁移，在预览版中可用。
  • MySQL的云SQL Insights】现已在预览版中推出。以前它只适用于PostgreSQL。
