谷歌云平台 (GCP)提供数十种 IaaS 、 PaaS 、 SaaS 服务。可悲的是，维基百科上关于GCP的条目是垃圾，虽然官方文件很好，但洒在上面的营销灰尘让我牙痛。作为我自己的参考，我对GCP上的每一项服务都做了客观的描述。

向 Greg Wilson 、Google Cloud开发者关系团队和Google Cloud开发者专家致敬，他们给出了四个字的描述。

计算

  • 计算引擎虚拟机、磁盘和网络一种 IaaS 服务，提供托管在谷歌基础设施上的虚拟机 (VMs)。竞争对手的服务包括亚马逊弹性计算云，以及 OpenStack 等本地对等服务。
  • App Engine【托管应用平台】一种 PaaS 服务，用于使用容器实例构建web应用和移动后端，这些容器实例预先配置有几个可用运行时中的一个，每个运行时都包括一组标准的App Engine库。竞争对手的服务包括亚马逊弹性豆茎和微软Azure网站。
  • 容器引擎托管库/容器码头工人容器的集群管理和编排系统。它基于开源的Kubernetes项目。
  • 容器注册表私有容器注册表&存储私有 Docker 存储库托管在谷歌的基础设施上。
  • 云功能 [ 无服务器微服务 ]一种基于事件的异步计算解决方案，允许您创建微服务(小型专用功能)，这些微服务可以响应云事件，而无需明确管理的服务器或运行时环境。谷歌云功能自2016年2月开始进入Alpha 。
  • Cloud Pub/Sub【分布式实时消息】一种完全托管的实时消息服务，用于在独立的应用程序之间发送和接收消息。
  • 用于应用引擎云API网关的 云端点框架，用于从您的代码中创建 RESTful 服务，并使其可供使用应用引擎的 iOS 、 Android 和 Javascript 客户端访问。原云端点。

存储和数据库

  • 云存储对象&文件存储和服务统一的对象存储服务，提供系列存储选项，包括地理冗余(低延迟、高 QPS 内容服务于跨地理区域的用户)、区域性(针对特定区域的工作负载)、近线(针对每月访问不到一次的数据)，以及)竞争对手的服务包括亚马逊简单存储服务(地理冗余/区域)和亚马逊冰川 (coldline)。
  • 云SQL【托管MySQL 】一种完全托管的 MySQL 数据库服务，用于在谷歌的基础设施上托管关系型MySQL数据库。
  • Bigtable[h base Compatible NoSQL]高性能 NoSQL 大数据数据库服务，旨在以一致的低延迟和高吞吐率支持超大型工作负载。Google 在内部使用Bigtable来支持包括搜索和Gmail在内的服务。
  • 云数据存储分布式分层键/值存储NoSQL无模式数据库，用于存储非关系数据。当需要 ACID 事务或者存储的数据高度结构化时，它是 Bigtable 的替代方案。
  • 云扳手 一个托管的全球分布式关系数据库，具有ACID事务、强一致性、SQL语义、水平伸缩、高可用性。
  • 持久磁盘 [ 可连接虚拟机的磁盘 ]一种提供SSD和HDD存储的服务，可以连接到在计算引擎或容器引擎中运行的实例。
  • 云源库托管私有Git库托管在GCP的私有Git 库；他们目前正处于测试阶段。

大数据

  • Big query无服务器数据仓储&分析S无服务器，全托管，Pb级数据仓库和分析平台，用于存储和查询大数据使用 SQL 。
  • 云数据流 [ 托管数据处理 ]全托管实时数据处理服务，用于批量和流式大数据处理，支持 ETL 、批量计算、连续计算。
  • data procManaged Spark&HadoopManagedApache Hadoop， Apache Spark ， Apache Pig ，以及 Apache Hive 服务用于处理大型数据集。
  • 云数据实验室【数据可视化与探索】一款基于 Jupyter (前身为IPython)构建的大规模数据探索、分析、可视化的交互工具。它支持使用 BigQuery 、计算引擎、使用 Python 、 SQL 、 JavaScript 的云存储。云数据实验室正在公测。
  • Google Genomics[Managed Genomics Platform]使用全球基因组与健康联盟定义的标准存储、处理、探索和共享基因组数据的API。这包括对管理数据集、读数和变量的支持；搜索和切片；以及为共享设置访问控制。

机器学习

  • 云机器学习 [ 带TensorFlow的机器学习 ]一个托管服务，用于使用 TensorFlow 框架构建机器学习模型。
  • 云视觉API【图像识别和分类】一个 REST API，可以用来将图像的内容理解成类别，检测图像中的单个对象和人脸，并查找和读取图像中包含的印刷文字 。
  • 云语音API将语音转换为文本RESTAPI，可用于将音频转换为文本。API可以识别超过80种语言和变体。谷歌云语音API目前正在公测。
  • 自然语言API【文本解析和分析】一个 REST API，可以用来解析文本的结构和含义。它可以在提供的文本中提取包括人物、地点、事件和情感在内的信息。谷歌云自然语言API目前正在公测。
  • 翻译API [ 语言检测和翻译 ]一个 REST API，可用于将任意语言字符串翻译成任何支持的语言。语言识别可用于未知源语言的情况。

联网

  • 谷歌云虚拟网络【软件定义网络】一套由谷歌管理的网络功能，包括粒度IP地址范围选择、路由、防火墙、虚拟专用网 (VPN)和云路由器，用于配置您的GCP资源，在虚拟私有云 (VPC)中将它们彼此连接和隔离。
  • 云负载均衡多区域负载分配负载均衡和自动扩展 GCP计算资源在单个 anycast IP 之后的单个或多个区域中的服务。
  • 云CDN [ 内容交付网络 ]利用Google全球分布的edge 存在点缓存HTTP(S) 负载均衡内容靠近用户。
  • 谷歌云互联与 GCP 对等允许GCP客户通过比他们现有的互联网连接更高可用性和/或更低延迟的连接来连接到谷歌。
  • 云DNS【可编程域名服务】一种托管的权威域名系统 (DNS)服务，运行在与Google相同的基础设施上。云DNS将对域名的请求翻译成 IP地址，并提供UI、命令行界面和API，用于发布和管理数百万个 DNS区域和资源记录。

身份和安全

  • Google Cloud IAM[身份&访问管理 ]让管理员授权谁可以对特定资源采取行动，以及内置的审计 。
  • 云资源管理器【云项目元数据管理】一种以编程方式管理资源容器(如组织和项目)的服务，用于分组和分层组织GCP资源 ⁴ 。
  • 云安全扫描器App Engine安全扫描器web安全扫描器针对 App Engine 应用中的常见漏洞，包括跨站点脚本 (XSS)、Flash注入、混合内容(HTTPS HTTP)、过时/不安全的库。

谷歌云平台管理服务

  • stack driver[云监控、日志记录&诊断为构建在云基础设施上的应用提供监控、日志记录和诊断，包括GCP和AWS。Stackdriver提供了度量、仪表板、警报、日志管理、报告和跟踪功能。
  • 部署管理器 [ 基于模板的基础设施部署 ]一种基础设施自动化和管理服务，允许您定义模板来部署各种GCP服务，包括云存储、计算引擎和云SQL 。 ⁵
  • 云壳基于浏览器的终端/CLI命令行从浏览器内访问云资源，无需在您的系统上安装谷歌云SDK或其他工具 ⁶ 。
  • Google cloud billing APIprogramatic GCP billing management以编程方式为您的GCP项目管理账单 ⁷ 。
