Cloud Spanner 是Google完全托管的可扩展关系数据库服务。我们最近发布了一个新特性，Spanner read statistics，它允许您运行SQL查询来检索数据库在1分钟、10分钟和60分钟间隔内的读取统计数据。这些读取统计数据允许您查看在您的数据库上执行的最常见和最消耗资源的读取(通过扳手读取API完成)。

在这篇文章中，您将看到如何使用这些读取统计数据来识别哪些读取涉及高CPU使用率。
