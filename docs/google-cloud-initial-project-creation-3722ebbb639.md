谷歌云平台为他们的许多产品提供了一个免费层。此外，他们还提供 300 美元的信贷。这是尝试他们提供的一切的好方法。

不过，要开始，你需要为免费层上的任何东西设置一个计费帐户，还需要设置你将使用各种服务的第一个项目。

在下面的短片中，我将介绍第一次设置和访问您的项目的过程。

我们会掩护

*   访问谷歌云控制台
*   创建项目
*   创建计费帐户
*   将您的帐单帐户链接到您的项目
