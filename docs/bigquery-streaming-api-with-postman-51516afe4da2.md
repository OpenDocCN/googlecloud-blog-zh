## 如何从 Postman 创建和执行 BigQuery 流请求

igQuery 是一个 Pb 级的无服务器数据仓库，带有一个内置的查询引擎。在传统数据仓库方法中，您提取、执行各种转换并将数据加载到数据仓库中。 [BigQuery](https://cloud.google.com/bigquery) 使用列存储格式，您可以简单地加载或流式传输您的数据并开始查询。您可以在几秒钟内查询万亿字节的数据。
