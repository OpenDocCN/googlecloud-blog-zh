Google Cloud Next’17

在我之前的帖子中，我承诺在接下来的几个月里谈论一些我正在参加或演讲的好会议。我最兴奋的会议之一是Google Cloud Next’17:Google的主要云会议将于3月8日至10日在三藩市举行。

去年，我以诺格勒的身份参加了那次会议。有很多开发者和很棒的技术含量。今年的日程刚刚公布，看起来更加令人兴奋，尤其是如果你是. NET开发者的话！

第一个，不要脸的塞。我将在Next’17发表演讲，我的主题是用谷歌云将你的ASP.NET应用提升到一个新的水平。我将谈论如何将现有的ASP.NET应用程序迁移到谷歌云，以及在谷歌云上运行你的ASP.NET应用程序会带来什么样的好处。这应该是一个信息丰富和有趣的谈话。NET开发人员。

我也对跑步感到兴奋。由乔恩·斯基特和克里斯·史密斯主持的谷歌云平台会议。本次会议将讨论如何将ASP.NET核心应用程序部署到应用程序引擎，并将 Kubernetes 部署到容器引擎。ASP.NET核心和Kubernetes在开发界都非常受欢迎，我很高兴谷歌云在很大程度上支持Kubernetes上的ASP.NET核心应用程序。

你可能不知道，但你可以在谷歌云上运行Windows Server和Microsoft SQL Server，在2017年的下一届会议上，这两个平台都将推出。在谷歌计算引擎上部署基于Windows的基础设施和在谷歌计算引擎上部署微软SQL Server应该都是了解所有细节的有趣会议。

除了那些伟大的窗户和。NET会话、无服务器架构(云功能)会话、机器学习、数据流大数据处理听起来都很有趣。更不用说，在Next我们将会有大量的代码实验室供人们亲身体验谷歌云。

作为一名. NET开发人员，我有很多理由对下一个2017年感到兴奋。希望能在那里看到你们中的一些人！
