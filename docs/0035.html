谷歌云平台有什么新功能？

  • 计算用户帐户(正式上市) — 通过简单安全的帐户和用户组管理对虚拟机的SSH访问。
  • 标签(全面上市) — 使用标签组织和管理您的Google云平台资源，可在API和开发者控制台中访问。
  • 指标变化率警报(全面上市) — 使用谷歌云监控的新功能，获得指标变化率而非静态阈值的通知。
  • 谷歌云安全扫描器(正式上市) — 主动测试您的App Engine应用程序是否存在许多常见的web应用程序安全漏洞。
  • 全新云管理控制台UI (测试版) — 通过锁定您最常用的服务来自定义您的控制台，体验改进的导航和搜索。
  • Google Cloud data lab(Beta)—借助围绕IPython构建的熟悉的数据科学生态系统，使用 Google BigQuery 和 Google Cloud Storage 的强大功能，快速、简单地做出数据驱动的决策。
  • Google Cloud Shell (测试版) — 从Google托管和管理的命令行访问您的Google云平台资源，并可在 Google开发者控制台中直接获得。
  • Google Cloud Monitoring和Google Cloud Logging中的新测试功能(测试版) — 根据日志数据识别趋势和问题，并通过基于日志的指标和警报更快地解决问题，这些指标和警报集成了 Google Cloud Logging 和 Google Cloud Monitoring 。
  • 应用引擎和云数据存储的新可用区域(测试版) — 将您的应用引擎、云数据存储和其他云平台服务放在南卡罗来纳州新的us-east1区域。
