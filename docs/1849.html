谷歌云和NVIDIA最近在Compute engine上推出了基于NVIDIA Ampere A100 Tensor Core GPU的A2 VMs。单个A2虚拟机利用由 NVIDIA的NVlink 结构支持的多达16个NVIDIA A100 GPU，在单个虚拟机中达到高达10 petaflops的FP16或10 petaflops的int8的有效性能。这种性能和规模是Google Cloud独有的，旨在满足当今最苛刻的HPC应用。

这种能力能够给 Altair 前所未有的周转时间，让他们用ultraFluidX进行CFD模拟。deep mind(Alphabet子公司)在利用机器类型之前，能够比以往任何时候都更快地训练他们的GPU实验。艾伦研究所能够在GCP上运行A100s，速度比他们现有的系统快4倍。点击了解更多关于A2 VM系列产品的信息。

构建ML应用程序是复杂的，但为生产中的应用程序提供服务却并非如此。谷歌云提供了简化机器学习操作(MLOps)的能力，以自动化ML生产生命周期。OpenX能够利用GCP工具来自动化其ML基础设施，并通过利用谷歌云平台上的独特功能来自动化其机器学习工作负载。 OpenX 优化了运行在Google Kubernetes引擎上的TensorFlow服务，在15毫秒内每秒可服务250万次预测请求。
